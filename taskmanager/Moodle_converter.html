📖 Предисловие

Добро пожаловать!

Это руководство поможет освоить Moodle XML Конвертер — программу для создания тестов в Moodle.

1
Набираете в Word

Так же, как для печатных тестов

2
Загружаете в конвертер

Программа сама определит типы вопросов

3
Импортируете в Moodle

Все вопросы появятся автоматически

💡 Возможности:
  • 7 типов вопросов
  • Автоопределение
  • Проверка ошибок
  • Предпросмотр

⬇️ Установка и скачивание

Скачайте последнюю версию с GitHub:

Скачать на GitHub
Для запуска понадобится:

Python 3.8+ и pip. Установка зависимостей:

pip install PyQt5 PyQtWebEngine lxml python-docx docxlatex

🖥️ Интерфейс

Описание элементов

1
Кнопки «Обзор»

Выбор файла Word (.docx) и папки для сохранения

2
Галочка «Выделить все»

Выделяет/снимает ВСЕ вопросы. Счётчик показывает выбрано/всего.

3
Дерево вопросов

Список вопросов с цветовой маркировкой по типу:

синий = выбор ответа
зелёный = ответ
оранжевый = сопоставление
красный = ошибка
4
Предпросмотр Moodle

Кликните на вопрос — увидите как будет в Moodle. Можно взаимодействовать!

🚀 Как пользоваться

Пошаговый процесс

1
Подготовьте файл Word

Создайте тест в Word по правилам разметки. Сохраните как .docx.

2
Запустите конвертер

Дважды кликните на Запустить конвертер.bat или введите python converter_gui.py

3
Выберите файл и папку

Нажмите «Обзор» для файла Word и выберите папку для сохранения XML.

4
Проверьте вопросы

Кликните на вопрос — увидите предпросмотр в Moodle. Красные = ошибки.

5
Экспортируйте

Нажмите «Экспорт» — получите XML-файл для импорта в Moodle.
